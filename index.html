Engenharia de Prompt na Prática

Um exemplo simples mostrando como a forma de escrever um prompt impacta diretamente a qualidade das respostas de uma Inteligência Artificial.

O que é Engenharia de Prompt?

Engenharia de prompt é a prática de estruturar instruções claras, contextualizadas e bem definidas para obter respostas mais úteis, precisas e alinhadas ao objetivo desejado ao interagir com modelos de IA.

Exemplo de Prompt Ruim ❌

Prompt
Me ajude a analisar dados.

Esse prompt é problemático porque não define contexto, objetivo, formato da resposta ou público-alvo.

Exemplo de Prompt Bom ✅

Prompt
Você é um analista de dados experiente.

Tenho uma base de dados de vendas com as colunas:
- data
- produto
- canal
- faturamento

Objetivo:
Identificar quais produtos tiveram queda de faturamento no último mês.

Formato da resposta:
- resumo em texto
- lista com os 3 produtos com maior queda
- sugestão de possíveis causas

Contexto:
Essa análise será apresentada para o time comercial.

Por que o prompt bom funciona melhor?

Elemento Prompt Ruim Prompt Bom
Papel da IA Não definido Analista de dados
Objetivo Vago Claro e específico
Contexto Inexistente Público e uso definidos
Formato Livre Estruturado

Checklist de um Bom Prompt

  • ✅ Definir o papel da IA
  • ✅ Explicar o contexto
  • ✅ Declarar o objetivo
  • ✅ Definir o formato da resposta
  • ✅ Informar o público-alvo
